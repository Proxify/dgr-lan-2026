# The Woodlands LAN 2026 - Art Style Guide

## Core Aesthetic

**Primary Style:** Synthwave/Retrowave meets Pixel Art Gaming
**Era Reference:** 1980s arcade culture, early 90s LAN parties, cyberpunk neon cities
**Mood:** Nostalgic, energetic, late-night gaming sessions, friendly competition

---

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Neon Cyan/Blue | `#00d4ff` | Primary accent, tech elements, UI highlights |
| Hot Pink/Magenta | `#ff0080` | Secondary accent, energy, action elements |
| Neon Green | `#00ff88` | Success states, nature/trees, health bars |
| Pixel Yellow | `#facc15` | Warnings, highlights, retro arcade feel |
| Deep Black | `#0a0a0a` | Backgrounds |
| Dark Purple | `#1a0a2e` | Secondary backgrounds, depth |

---

## Visual Elements

- **Grid patterns** - Perspective grids fading into horizon (Tron-like)
- **Scanlines** - CRT monitor effect overlay
- **Glitch artifacts** - Digital distortion, RGB split
- **Neon glow** - Bloom effects on text and objects
- **Pixel art elements** - 8-bit/16-bit style icons and characters
- **Geometric shapes** - Triangles, hexagons, circuit patterns
- **Palm trees & sunsets** - Classic synthwave silhouettes
- **Gaming hardware** - Controllers, monitors, keyboards, mice

## Implementation Notes

### File Naming Convention
```
twl2026-[section]-[descriptor]-[size].png
Examples:
- twl2026-hero-background-1920x1080.png
- twl2026-squad-frame-400x500.png
- twl2026-icon-controller-128x128.png
```

### Optimization Requirements
- All images should be compressed for web (TinyPNG or similar)
- Create WebP versions for modern browsers
- Hero/large backgrounds: max 200KB
- Icons/small elements: max 50KB
- Consider creating 2x versions for retina displays

### CSS Integration Tips
- Use `mix-blend-mode: screen` or `overlay` for neon effects
- Apply subtle `filter: hue-rotate()` for color variations
- Layer with gradient overlays to maintain text readability
- Use `mask-image` for fade effects at edges
